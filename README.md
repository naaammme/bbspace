# bbspace
<p align="center"><img src="images/banner.jpg" alt="bbspace banner" width="480" /></p>

一个使用 Jetpack Compose + Material3 开发的第三方 B 站 Android 客户端。

## 已实现功能

- 扫码登录 / 短信登录
- 首页推荐视频流
- 视频播放
- 多账户管理
- 设置系统
- md3
- 其他
