  一个使用 Jetpack Compose 和 Material 3 构建的第三方 B 站 Android 客户端。
</p>

## 下载

前往 [Releases](https://github.com/naaammme/bbspace/releases) 页面获取最新安装包。

## 已实现功能

- 扫码登录和短信登录
- 首页推荐视频流
- 视频播放
- 多账号管理
- 设置
- 视频搜索
- 其他基础能力持续补全中
